# Security Policy

## Scope

This policy applies to **kaynos-docs**, the Kaynos Help Center static site
(published at [docs.kaynos.net](https://docs.kaynos.net)).

## Reporting a vulnerability

If you discover a security issue in this repository, please report it
responsibly:

- **Email:** security@kaynos.net
- **Response:** We will acknowledge your report within **48 hours** and
  provide an initial triage within **7 business days**.

Please include:
1. A description of the vulnerability.
2. Steps to reproduce (URL, browser, screenshot if helpful).
3. The potential impact as you understand it.

## Out of scope

The **main Kaynos application** (app.kaynos.net) is maintained in a separate
repository. If the issue affects the application rather than this
documentation site, please report it to the
[kaynos repository](https://github.com/jlambert229/kaynos) instead.

## Safe harbor

We support responsible disclosure. If you act in good faith and follow
this policy:

- We will not pursue legal action against you.
- We will work with you to understand and resolve the issue promptly.
- We will credit you (unless you prefer to remain anonymous) if we
  publish an advisory related to your report.

We ask that you:
- Make a good-faith effort to avoid privacy violations, data destruction,
  and service disruption.
- Only interact with accounts you own or have explicit permission to test.
- Give us a reasonable window to address the issue before public disclosure.
